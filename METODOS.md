```
 * clear() - Este método é usado para limpar todas as chaves existentes dentro do HashMap.
 * put(chave, valor) - Este método é usado para armazenar uma chave com um valor dentro do HashMap.
 * putIfAbsent(chave, valor) - Este método é usado para armazenar uma chave e o valor no HashMap somente se a chave não existir.
 * size() - Este método é usado para retornar o número de chaves existentes no HashMap.
 * containsKey(chave) - Este método é usado para verificar se há uma chave específica no HashMap.
 * containsValue(valor) - Este método é usado para verificar se há um valor específico no HashMap.
 * isEmpty() - Esté método retorna se o HashMap está vázio. (True para sim, e False para não)
 * values() - Este método retorna uma coleção de valores do HashMap.
 * remove(chave) - Este método remove uma chave específica do HashMap.
 * remove(chave, valor) - Este método remove uma chave específica do HashMap, porém somente se a chave possuir o valor específicado.
 * get(chave) - Este método retorna o valor da chave encontrada dentro do HashMap.
 * replace(chave, valor) - Este método substitui o valor da chave específica somente se ela existir dentro do HashMap.
 * replace(chave, antigoValor, novoValor) - Este método substitui o valor da chave específica, mas a chave deve possuir o valor antigo, para que o novo seja sobreposto.
 * forEach(key, value) - Este método te retorna todos os mapeamentos do HashMap, retornando como parâmetros a Chave e o Valor.
```
